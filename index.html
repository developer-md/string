print method

print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)


print() Parameters:-

objects - object to the printed. *indicates multiple objects

sep - objects are separated by sep. Default value space.

end - end is printed at last

file - default,sys.stdout will be used which prints objects on the screen.

flush - If True, the stream is forcibly flushed. Default value: False

Example of file parameter:-

sourceFile = open('python.txt', 'w')

print('Pretty cool, huh!', file = sourceFile)

sourceFile.close()
